# ClassIsland Marketplace

A web marketplace preview for ClassIsland plugins. This Next.js application provides a modern, responsive interface for discovering and downloading ClassIsland plugins.

## Features

- **Cards Layout**: An aesthetically pleasing, responsive layout for browsing available plugins.
- **Fluent UI Design**: Integrated with Microsoft's Fluent UI design system for a native, polished look consistent with Windows environments.
- **Dynamic Plugin Data**: Real-time fetching of plugin manifests and data from the ClassIsland PluginIndex repository on GitHub.
- **Cross-Platform Delivery**: Smart "Install" and "Download" buttons that adapt based on the user's operating system natively supporting `.cipx` extensions.
- **Markdown Previews**: Detailed plugin descriptions rendered richly with `@uiw/react-markdown-preview`.

## Tech Stack

- Next.js (React)
- Fluent UI React Components
- TypeScript
- Tailwind CSS

## Getting Started

1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
